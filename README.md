# NASberryPI - A Raspberry PI 5 NAS System
NASberryPI software

<img width="620" height="733" alt="Screenshot 2025-11-09 at 7 36 14 PM" src="https://github.com/user-attachments/assets/d15263a2-2fc8-4136-899b-6af1c2aaec7d" />

NASberryPI is a network-attached storage (NAS) solution built on the Raspberry Pi 5 platform. It leverages USB 3.0 Seagate drives—or any compatible USB 3.0 external storage device—to create a compact yet powerful personal storage server.
The software included in this repository manages and controls the system’s display interface, providing users with the following functionality:
Real-time system monitoring (e.g., CPU usage, disk activity, and network throughput)
Storage management and status visualization

Connection and access indicators for network and drive status

Customizable display themes or information layouts
* weather reports
* CPU temperature
* date/time
* Memory utilization

Turn the Knob on the front display and you can get:
* SMART reporting
* Reboot/Shutdown controls
* Game of Life
* Space Invaders
* Restart of Local Services
* Plus additional functions
